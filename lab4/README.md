Выполняется уже не в VS code, а в Gowin. Логика создаётся в файле top.v, список подключений к внешним портам -- в файле lect.cst. 

Чтоб использовать кнопку S0 на плате, надо было в Project -> Configuration -> Dual purpose pin отметить галочку Use SSPI as regular IO.

Если хотим посмотреть графики временных процессов, надо созадть .gao файл, добавить туда таймер. Тактовую частоту можно брать с Oscillator на H11, 27MHz. Таймеры ещё надо будет добавлять в time constraints и делить частоту хотя бы счётчиком.
